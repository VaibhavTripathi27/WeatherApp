To run the project.....
Make sure you have node and npm installed in your system.
Download the source code files and using terminal navigate towards the folder
run command ```npm init```... to initialise the npm in your project
After above step run command ```npm i express``` and ```npm i body-parser``` to use express and body-parser as dependencies in the code
Get your own API Key from the https://api.openweathermap.org/ and paste it inside the apiKey in app.js
after above mentioned step run ```node app.js```.
